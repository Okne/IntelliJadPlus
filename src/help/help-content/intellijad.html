IntelliJad Plus
IntelliJad Plus is a plugin for IntelliJ IDEA that provides a wrapper for the Java decompiler Jad, plus a set of management options that make it easy to work with inside an IDE.
All the credit for plugin creation goes to Steve Chaloner
